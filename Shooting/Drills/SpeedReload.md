# Speed Reload

## Ammo present

* Pew pew pew
* Weak hand to new mag
* Eject mag with a flick motion
* Insert new mag
* Pull magazine to check if it is seated properly
* Pew pew pew

## No ammo present

* Pew pew click click
* Check gun, chamber empty
* Weak hand to new mag
* Eject mag with a flick motion
* Insert new mag
* Pull magazine to check if it is seated properly
* Rack slide
* Pew pew pew
